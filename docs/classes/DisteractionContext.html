    Class DisteractionContext<E>

    Enhanced context class for handling Discord interactions in serverless environments.

    This class wraps the Hono context and provides additional functionality specific to Discord bot interactions. It manages the Discord API client, interaction handlers, and provides convenient access to environment variables and request data.

    The context automatically creates lookup maps for efficient interaction routing and provides type-safe access to all registered commands, buttons, and modals.

    Example

    const context = new DisteractionContext({
      hono: c, // Hono context from request handler
      discordToken: c.env.DISCORD_TOKEN,
      discordPublicKey: c.env.DISCORD_PUBLIC_KEY,
      commands: [pingCommand],
      buttons: [counterButton],
      modals: [feedbackModal]
    });
