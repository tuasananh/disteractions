    Class MessageComponentButtonInteraction<E>

    Base class for all Discord interactions.

    This class provides common functionality for all interaction types including type guards, response methods, and access to the interaction context and data. It serves as the foundation for more specific interaction classes.

    Example

    // Type guards for interaction handling
    if (interaction.isApplicationCommand()) {
      // Handle application command
    } else if (interaction.isMessageComponent()) {
      // Handle button/select menu
    }
